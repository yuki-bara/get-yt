## check version

```bash
get-yt -V
# Version: 1.3
```

## download video

```bash
get-yt -mp4 https://www.youtube.com/watch?v=123456789ab
#or
get-yt -mp4 123456789ab
```

But you won't get the best video quality.

[download videod (upgrade)](#download-video-upgrade)

## download sound

```bash
get-yt -m4a https://www.youtube.com/watch?v=123456789ab
#or
get-yt -m4a 123456789ab
```

## download video (upgrade)

What is mp4u?

This process combines m4a and m4v files to achieve the highest video quality.

```bash
get-yt -mp4u https://www.youtube.com/watch?v=123456789ab
#or
get-yt -mp4u 123456789ab
```

## clean cache

```bash
get-yt -C
```
