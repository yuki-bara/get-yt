# how-to-use

- [check version](#check-version)
- [download video](#download-video)
- [download sound](#download-sound)

## check version

```bash
get-yt -V
# Version: 1.3
```

## download video

```bash
get-yt -mp4 https://www.youtube.com/watch?v=123456789ab
#or
get-yt -mp4 123456789ab
```
## download sound

```bash
get-yt -m4a https://www.youtube.com/watch?v=123456789ab
#or
get-yt -m4a 123456789ab
```
