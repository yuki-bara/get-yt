GET-YT (**GET-Y**ou**T**ube-Video) Is program downloads videos from YouTube (CLI)

## INSTALL

1. Clone the GET-YT repository
```bash
git clone https://github.com/yuki-bara/get-yt.get
cd get-yt
```

2. Create a binary file
```bash
make build
```

Now, have a binary file in bin folder
