# GET-YT

[![Go](https://img.shields.io/badge/Go-1.26.1%2B-3776AB?logo=Go&logoColor=white)](https://go.dev/)
[![License: 0BSD](https://img.shields.io/badge/License-0BSD-yellow.svg)](./LICENSE)

GET-YT (**GET-Y**ou**T**ube-Video) Is program downloads videos from YouTube (CLI)

## INSTALL

install

```bash
curl -fsSL https://github.com/yuki-bara/get-yt/raw/refs/heads/main/installation/auto_install.bash | bash
```

check

```bash
get-yt -V
```
